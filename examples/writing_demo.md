# 實戰操作：撰寫文章 (Writing Articles)

## 情境
不只是程式碼與技術文件，你還需要撰寫各種文案，例如：
- **科技文章**：介紹新技術趨勢或 Gadget 開箱體驗。
- **學習筆記**：整理複雜的論文或線上課程摘要。
- **公文/報告**：撰寫正式的會議記錄、提案書或行政公文。

---

## Antigravity 做法 (Agentic Workflow)

### 1. 定義風格與指令
利用 **Skill** 定義特定的寫作風格，例如「輕鬆幽默」的部落格風格，或是「嚴謹正式」的公文風格。

> "請閱讀我附上的這篇 PDF 論文摘要，幫我整理成一份『學習筆記』。
> - 使用繁體中文。
> - 結構包含：核心論點、實驗結果、個人心得。
> - 使用 LaTeX 格式撰寫數學公式。"

### 2. 多階段寫作 (Iterative Writing)
Agent 會按步驟協助你完成：

1.  **大綱規劃**：先產出文章架構供你確認。
2.  **內容填充**：根據大綱撰寫初稿。它還可以自動上網搜尋相關數據來佐證論點。
3.  **潤飾優化**：檢查流暢度、錯別字，並依照你的要求調整語氣。
